## DailySMP joinen met de Java editie van Minecraft

<Steps>
  <Step title="Download en installeer eerst Minecraft">
    Mocht je dit nog niet hebben.
  </Step>
  <Step title="Log-in met een Microslop-Account">
    Zorg er voor dat op het account Minecraft Bedrock of Java aangekocht is.
  </Step>
  <Step title="Start minecraft op">
    Dit kun de door op de play-knop te drukken

    <Warning>
      Zorg er voor dat je minecraft-versie hoger is dan 1.21.0. Alle versies daaronder worden niet toegestaan op de server
    </Warning>
  </Step>
  <Step title="Klik op de knop 'Multiplayer'">
    Doe dit zodra Minecraft opgestart is.
  </Step>
  <Step title="Klik op de knop 'Add Server'">
    Voer dan de volgende gegevens in:

    <Frame>
      ![Screenshot 2026 04 26 232943](/images/Screenshot-2026-04-26-232943.png)
    </Frame>
  </Step>
  <Step title="Klik op 'Done'">
    Dan komt de server te staan in de lijst met servers.
  </Step>
</Steps>
